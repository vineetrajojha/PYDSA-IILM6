# PYDSA-IILM6

A course repository for **Python + Data Structures & Algorithms**, maintained for the IILM Batch 6 cohort. It contains class notebooks, LeetCode-style Python solutions, SQL query exercises, and a curated DSA practice sheet.

## 🧠 Topics Covered

### Data Structures & Algorithms (Python)

| Problem | Key Concept |
|---|---|
| `asteroidCollision.py` | Stack simulation |
| `findKthBitinNthBinaryString.py` | Recursion / String manipulation |
| `implementQueueUsingStacks.py` | Stack → Queue using two stacks |
| `longestCommonPrefix.py` | String comparison |
| `longestSubstringWithoutRepeatingCharacters.py` | Sliding window / HashMap |
| `nextGreaterElement.py` | Monotonic stack (optimised) |
| `nextGreaterElementBrute.py` | Brute-force O(n²) approach |
| `partitioningIntoMinimumNumberDeciBinary.py` | Greedy |
| `rotateList.py` | Linked list manipulation |
| `validParanthesis.py` | Stack-based bracket matching |

### SQL

| File | Problem |
|---|---|
| `customersWhoNeverOrder.sql` | `LEFT JOIN` / `NOT IN` subquery |
| `duplicateEmails.sql` | `GROUP BY` + `HAVING COUNT > 1` |
| `deleteDuplicateEmails.sql` | `DELETE` with self-join |

## 🚀 Getting Started

**Prerequisites**

- Python 3.8+
- Jupyter Notebook / JupyterLab
- Any SQL client (MySQL, SQLite, etc.)

**Run a Python solution**

```bash
python asteroidCollision.py
```

**Open a class notebook**

```bash
jupyter notebook feb_17.ipynb
```
